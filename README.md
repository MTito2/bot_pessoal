# Bot Pessoal no Telegram

Projeto desenvolvido com o objetivo de centralizar e automatizar controles pessoais utilizando programação, inteligência artificial e integrações com APIs externas.

A ideia principal foi aplicar a programação como ferramenta de resolução de problemas reais do dia a dia, criando uma solução modular, reutilizável e preparada para crescimento.

---

## Objetivo do Projeto

Centralizar, em um único bot no Telegram, o controle de:

* Finanças pessoais
* Estudos
* Organização de agenda
* Alimentação

Tudo isso de forma automatizada, simples de usar e com apoio de IA para interpretação de dados e linguagem natural.

---

## Funcionalidades

* Análise de imagem de cupom fiscal utilizando IA
* Registro de gastos
* Registro de estudos
* Conversão de período em linguagem natural para data (ex: "última semana", "mês passado") usando IA
* Métricas financeiras mensais
* Métricas de estudo mensais
* Sugestão de dieta baseada em IA
* Criação de eventos diretamente no Google Agenda

---

## Arquitetura e Organização

O projeto foi desenvolvido com foco em:

* Estrutura modular
* Separação clara de responsabilidades
* Facilidade de manutenção
* Possibilidade de reutilização e escalabilidade

Cada funcionalidade foi pensada como um módulo independente, facilitando futuras expansões ou adaptações.
